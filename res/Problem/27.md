# 结构体之成绩统计2


## 题目描述

有N个学生，每个学生的数据包括学号、姓名、3门课的成绩，从键盘输入N个学生的数据，要求打印出3门课的总平均成绩，以及最高分的学生的数据（包括学号、姓名、3门课成绩）

## 输入格式

学生数量N占一行每个学生的学号、姓名、三科成绩占一行，空格分开。

## 输出格式

各门课的平均成绩 最高分的学生的数据（包括学号、姓名、3门课成绩）

## 数据范围

N<100

## 样例

### 输入样例

```
2
1 blue 90 80 70
b clan 80 70 60
```

### 输出样例

```
85 75 65
1 blue 90 80 70
```
