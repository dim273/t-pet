# 结构体之成绩记录


## 题目描述

现有有N个学生的数据记录，每个记录包括学号、姓名、三科成绩。 编写一个函数input,用来输入一个学生的数据记录。 编写一个函数print,打印一个学生的数据记录。 在主函数调用这两个函数，读取N条记录输入，再按要求输出。 

## 输入格式

学生数量N占一行 每个学生的学号、姓名、三科成绩占一行，空格分开。

## 输出格式

每个学生的学号、姓名、三科成绩占一行，逗号分开。

## 数据范围

N<100

## 样例

### 输入样例

```
2
a100 clang 70 80 90
b200 dotcpp 90 85 75
```

### 输出样例

```
a100,clang,70,80,90
b200,dotcpp,90,85,75
```
